# Recognition Gap EP 7層ランキング仕様

## 入力

- 米国株ユニバース: `stock.csv` またはFMP stock screenerから取得したNASDAQ/NYSE銘柄
- 日足OHLCV: `data/price_data_ohlcv.pkl`
- 企業属性: FMP profileまたは`stock.csv`内の `CompanyName / Sector / Industry / MarketCap / Country`
- 任意: ニュース、決算、受注、バックログ、保有者データ、米10年金利

## ライブランキングで使ってよい情報

`asof_date` 以前に取得可能な情報だけを使う。将来高値、将来ニュース、将来決算、将来のテーマ分類は使わない。

## 入口候補

基本構成:

```text
entry_rule: industry_theme_ep_ex_biotech
entry_timing: pullback10
exit_rule: stage2_or_atr8
```

抽出条件:

- バイオ/医薬イベント銘柄を原則除外
- 低流動性を除外
- 価格が10日・20日移動平均線を上回り、20日または60日で強い変化を持つ
- 市場比の60日超過分が弱すぎない
- 出来高またはドル出来高の増加がある
- EP後、10日移動平均線接近または維持による日足入口を採用

## ランキング

ランキングは機械的な優先順位で作る。LLMに数値スコアを任せない。

優先するもの:

- `thesis_intact`
- `price_trend` が `strong` または `extended_but_intact`
- `volume_demand_durability` が `durable_accumulation`
- 株式需給リスクが低い
- `ret60_resid_spy` が強い
- `post_signal_dv_persistence` が高い
- 最低限のドル出来高がある

## 7層

1. データ整合性: 銘柄、会社名、業種、ニュース混入を確認
2. 価格トレンド: 移動平均線、上昇局面、伸び切りでも崩れていないか
3. 出来高・資金流入の持続: EP後の出来高維持、ドル出来高、上昇日出来高比
4. 株式需給リスクと流動性: 増資、売り出し、ワラント、転換社債、大株主売り、時価総額、売買代金
5. 材料の質: 業種再評価、決算、受注、受注残、ニュース文脈
6. 業績・受注確認: 売上、1株利益、受注、受注残、構造変化
7. 見立て状態: 見立てが保たれているか、強弱混在か、傷んでいるか

`thesis_state` は毎日ゼロから揺らさず、実運用では前日状態、`daily_flags`、重大イベント、3営業日継続を使って昇格/降格させる。

## opencode go合議

4モデルの役割:

- Kimi: 銘柄エンティティ、ニュース混入、事実確認
- MiniMax: 価格、出来高、株式需給リスク、事業需給・受注の違和感
- GLM: 業種、競合、構造テーマ、ファンダ文脈
- DeepSeek: まとめ役。各モデルの意見を統合して日本語要点を作る

出力は売買判断ではなく、300字以内の要点、注意点、銘柄一致確認、見立ての補助分類に限定する。

## X投稿

- 画像は白黒テーブル
- 検出された全銘柄を5銘柄ずつ分割する。例: 1-5、6-10、11-15。最後は余り件数でよい
- 表示順は含み益順ではなく、7層の優先保有/監視順にする
- 列: `順位 / 銘柄 / 入口日 / 含み益 / 要点`
- 本文は原則 `$TICKER` を20銘柄分だけ並べる。内部検証では全ティッカー出力も許可する
- 要点はニュース見出しの貼り付けではなく、価格、出来高、業種文脈、業績、株式需給リスク、事業需給・受注、見立て状態を統合した300字以内の日本語にする
- 要点は全銘柄に同じ層を並べない。売上前年比、売上前期比、1株利益の前年比/前期比、受注、受注残、契約、機関投資家買い、時価総額リスクなど、その銘柄だけの強い証拠を優先する
- 1株利益は単体の数値をそのまま載せない。前年比、前期比、前回からの加速/減速/改善/悪化が取れる場合だけ書く
- `supply_risk_severity` は内部値として `low / medium / high` のいずれかを原則付与するが、表示では `低い / 中程度 / 高い` に変換する。欠損時は時価総額、ドル出来高、増資、売り出し、ワラント、転換社債、包括登録、二次売出し系ニュースから推定する

## 禁止

- Webull注文機能
- 売買ボタン
- 生成AIによる直接の出口判断
- 先見バイアス
- ニュースと別会社ティッカーの混入
